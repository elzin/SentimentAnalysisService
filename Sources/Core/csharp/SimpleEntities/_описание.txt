Файл описания сущностей:
1 Корневым тегом является <root>
2 Тег описания сущности <entity>
  2.1 Атрибут Type - тип сущности. Доступные символы: "A-Za-zА-Яа-я0-9_"
  2.1 Атрибут Name - название типа сущности. Если данного атрибута нет, то названием типа выступает значение атрибута Type.
  2.3 Атрибут dictionary0:dictionary9 - имя файла словаря специальной лексики
  2.4 Атрибут Select - уровень выделения сущности. Если атрибута нет, то сущность будет всегда выделяться.
    2.4.1 False - не выделять сущность.
    2.4.2 Child - выделять только как дочернию сущность.
    2.4.3 True - всегда выделять сущность.
3 Тег описания регулярного выражения для сущности <regex>
  Между открывающим и закрывающим тегами располагается само регулярное выражение.
  Так же в регулярных выражениях возможно использовать ранее определенные с помощью следующей конструкции:
  (@entity_type), гда entity_type - тип используемого регулярного выражения.
  В случае необходимости использования символа @ требуется заключить его в двойные кавычки: "@".
  Для того, чтобы в качестве значение было выбрано не все регулярное вырадение, а только его часть, требуется использовать следующую конструкцию:
  (?<value>regex), где regex - регулярное выражение для выделения значения сущности
